# E-Commerce System 🛒

This is a simple console app built in C# for the **Fawry Internship Challenge**.

## Features

- Add products with name, price, quantity
- Handle products that can expire or need shipping
- Add items to a cart with quantity checks
- Checkout with:
  - Subtotal
  - Shipping fees
  - Total amount
  - Updated customer balance
- Show shipped items and their total weight
